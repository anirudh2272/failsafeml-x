# GitHub Release Checklist

Before sharing publicly:

- [ ] `python -m pytest` passes.
- [ ] M1 through M8 scripts run successfully.
- [ ] `.env` is not committed.
- [ ] `.env.example` is committed.
- [ ] README describes the completed project, not only an early milestone.
- [ ] Generated reports are included or reproducible.
- [ ] Screenshots are placed under `assets/screenshots/`.
- [ ] The repository does not claim production safety certification or patentability.
